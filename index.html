themer takes a set of colors and generates text editor themes, terminal themes, and desktop/device wallpapers.

Download for macOS, Linux, or Windows
Themer on Product Hunt:

Remix built-in themes or create your own from scratch.

Built-in themes:

  • Default
  • Night Sky
  • One
  • Polar Ice
  • Lucid
  • Finger Paint
  • Solarized
  • GitHub Universe
  • More coming soon...

Use your favorite theme across all your apps.

Supported apps so far:

  • Hyper.app
  • iTerm
  • Terminal.app
  • Atom Syntax
  • Atom UI
  • Sublime Text
  • Vim
  • lightline.vim
  • VS Code
  • Xcode
  • BBEdit
  • Slack (Sidebar)
  • Alfred.app
  • ConEmu
  • CMD.exe

Generate beautiful desktop wallpapers that match your theme.

  • “Block Wave”
  • “Octagon”
  • “Triangles”
  • More coming soon...

Prefer the command line? Integrate themer's CLI into your dotfiles repository to programmatically generate themes.
